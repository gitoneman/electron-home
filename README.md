## how to run
首先确保你已经全局安装了electron

 1. git clone git@github.com:gitoneman/electron-home.git
 2. cd electron-home && electron main.js

## 还有一个window可执行文件
进入electron-v0.35.0-win32-x64这个目录，双击执行electron.exe也可以运行
